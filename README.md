# Article Knowledge Base Application
This is a knowledgebase app built with "Node.js & Express"

## Technologies 
* Node.js
* Express
* Express Messages, Session, Connect Flash & Validation
* MongoDB & Mongoose
* Pug Templating
* Passport.js Authentication
* BCrypt Hashing

## Version
1.0.0

## Usage
### Installation
Install the dependencies
##### $ npm install
Run app
##### $ npm start
